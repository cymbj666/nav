# 🧭 我的导航

个人导航页 —— 把常用网站和工具集中在一个页面上，随时打开就能用。

## 功能

- **卡片网格布局** —— 按分类排列，一目了然
- **实时搜索** —— Ctrl+K 聚焦搜索框，跨名称、描述、标签匹配
- **深色模式** —— 自动跟随系统，也可手动切换
- **Markdown 教程弹窗** —— 卡片内嵌教程按钮，点击弹出渲染后的教程
- **响应式** —— 手机、平板、桌面都能用
- **零依赖离线可用** —— 纯静态页面，下载即用

## 技术栈

纯静态 HTML + CSS + JavaScript，托管在 GitHub Pages。

- 字体：霞鹜文楷（LXGW WenKai）
- Markdown 渲染：marked.js

## 本地使用

直接双击 index.html 即可在浏览器打开。

## 添加链接

编辑 index.html，在 window.__NAV_LINKS__ 数组中按格式添加条目。

可选字段：extraButton 用于卡片内嵌按钮（教程弹窗或外链跳转），tags 用于搜索关键词。
